TopMSV Server Setup Guide

1. Install Node.js

  1. Go to the Node.js website
  2. Click download button to download the installation file. The version number may be different.

  3. Double click the downloaded file "node-v14.17.0-x64.msi" to install it. (The numbers may be different)

  4. Follow the instructions of the setup wizard to finish installation.

2. Start TopMSV Server

  1. (First Time Only) Right click "installServer.bat" and select "Run as administrator."

    2. When the messages below are displayed, proceed to the next step.

  2. Double click "startServer.bat". From the second time, you only need to run this file to use TopMSV Server. A chrome browser will open when TopMSV Server is ready.

  3. When finished using TopMSV Server, double click "stopServer.bat" to close TopMSV Server.
