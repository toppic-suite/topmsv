TopMSV Server Setup Guide

1. Install Node.js

  1. Go to the Node.js website
  2. Click download button to download the installation file.

  3. Double click the downloaded file "node-v14.17.0-x64.msi" and follow the instructions of the setup wizard to finish installation.

    4. When asked "Do you want to allow this app to make changes to your device?", select "Yes".

2. Start TopMSV Server

  1. Double click "startServer.bat". A chrome browser will open when TopMSV Server is ready.

  2. If a pop-up below is displayed, click "Open."

  3. When finished using TopMSV Server, double click "stopServer.bat" to close TopMSV Server.
