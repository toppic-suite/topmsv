How to Start TopMSV Server

1. Start TopMSV Server

  1. Double click "scripts" folder

  2. Double click "windows" folder

  3. Double click "startServer.bat". A chrome browser will open when TopMSV Server is ready.

  4. If a pop-up below is displayed, click "Open."

2. Stop TopMSV Server

  1. Double click "scripts" folder

  2. Double click "windows" folder

  3. Double click "stopServer.bat" to close TopMSV Server.
